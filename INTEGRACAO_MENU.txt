## PASSO 3: Crie uma NOVA funcao no FINAL do arquivo

Copie e cole isto no FINAL do arquivo user_controller.py:

```python
# Controller para menu de energia
def menu_energia_controller(user):

    # while = loop infinito ate break
    while True:

        print("\n===== MENU DE ENERGIA =====")
        print("1 - Consumo simples")
        print("2 - Classificacao")
        print("3 - Economia simples")
        print("4 - Opcoes de economia")
        print("0 - Voltar")

        opcao = input("\nEscolha opcao: ")

        # if/elif/else para cada opcao
        if opcao == "1":
            mostrar_consumo_simples(user)

        elif opcao == "2":
            mostrar_classificacao(user)

        elif opcao == "3":
            mostrar_economia_simples(user)

        elif opcao == "4":
            mostrar_opcoes_economia(user)

        elif opcao == "0":
            break

        else:
            print("Opcao invalida")
```

---

## PASSO 4: Modifique a funcao user_menu_controller()

Procure a funcao `user_menu_controller()` que tem este trecho:

```python
# Consumo energetico
elif option == "3":
```

Mude para:

```python
# Consumo energetico
elif option == "3":
```
